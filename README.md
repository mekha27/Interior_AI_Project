# Interior AI Project 🏠🤖

AI-Powered Interior Object Detection System built using Django REST Framework and YOLOv8.

## 🚀 Features

- Object Detection using YOLOv8
- Django REST API integration
- Image upload and detection output
- PostgreSQL ready
- Clean project structure

## 🛠 Tech Stack

- Python
- Django REST Framework
- YOLOv8
- OpenCV
- PostgreSQL

## 📂 Project Structure

- detection/ → Object detection app
- interior_ai/ → Django project settings
- manage.py → Django entry point

## ⚙️ Setup Instructions

```bash
git clone https://github.com/mekha27/Interior_AI_Project.git
cd Interior_AI_Project
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py runserver
